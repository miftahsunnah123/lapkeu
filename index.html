Kesalahan Konfigurasi Firebase

Aplikasi tidak dapat terhubung ke database karena metode "Sign-in Anonymous" belum diaktifkan di proyek Firebase Anda.

Langkah Perbaikan:
Buka Firebase Console, masuk ke bagian Authentication → Sign-in method, dan aktifkan provider 'Anonymous'.

Kode Error:

Izin Akses Ditolak

Aplikasi tidak memiliki izin untuk membaca atau menulis data. Ini disebabkan oleh Firestore Security Rules yang terlalu ketat.

Langkah Perbaikan:
Buka Firebase Console, pilih proyek Anda, lalu ke Firestore Database → Rules, dan ganti aturannya dengan kode di bawah ini:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /laporan-keuangan/{year}/{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
